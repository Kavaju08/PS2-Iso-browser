# PS2-Iso-browser

A Python tool for managing and browsing PlayStation 2 game backups. This project stems from the need to organize ISO libraries more efficiently than a conventional file explorer.

### ⚠️ Note: Under development
The project is in an early stage. I'm currently working on the scanning logic and database structure for metadata.

### Current / Planned Features
- [x] Initial project structure.
- [ ] File scanner script for `.iso` and `.bin` files.
- [ ] Game ID extraction (e.g. SLUS-21605).
- [ ] Interface to display cover art and official game titles.
- [ ] Integration with external PS2 title databases.

### Prerequisites
* Python 3.10 or higher.
